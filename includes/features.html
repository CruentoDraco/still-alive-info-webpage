Ihr fragt euch was Still Alive für ein Spiel wird? Dann seid ihr hier richtig. Still Alive wird ein Base Building Survival Strategie Spiel.

Sprich ihr werdet eine Basis aufbauen die immer größer wird. Und müsst täglich gegen Zombiehorden bestehen.

Ein Early Access Release ist im Herbst 2022 geplant.

Welche Game Features sind für den Early Access geplant? Für den Early Access sind folgende Features geplant:

  • Base Building zum Beispiel Gebäude bauen oder die Basis erweitern
  • Handelsfunktion mit anderen Fraktionen
  • Dynamischer Tageswechsel
  • Außenposten Funktion (Lite Version)
  • Erkundungstouren (Lite Version)
  • Personenverwaltung (Lite Version)

Wenn ein Feature Lite Version in Klammern hat bedeutet dies, das die Funktion in abgespeckter Funktion hinzugefügt wird. Diese Funktionen erhalten aber über die Jahre immer mehr Funktionen oder Verbesserungen
